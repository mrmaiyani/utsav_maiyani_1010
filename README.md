# 🌐 Data Science Portfolio Website

A **professional, responsive portfolio website** designed to showcase data science skills, projects, and technical expertise. Built using **HTML, CSS, and JavaScript**, this portfolio highlights real-world capabilities in data analytics, visualization, and frontend development.

This project is ideal for **placements, internships, and personal branding**.

---

## 🎯 Project Objective

To create a clean and modern portfolio that:

* Demonstrates technical and analytical skills
* Showcases projects effectively
* Provides recruiters an easy way to connect

---

## 🚀 Key Features

* Responsive design (mobile-first approach)
* Modern UI with smooth animations and transitions
* Skills section with dynamic progress indicators
* Project showcase with filtering functionality
* Animated statistics counters
* Functional contact form using Formspree
* Clean navigation with smooth scrolling

---

## 🧠 Skills Demonstrated

* Frontend Development (HTML, CSS, JavaScript)
* UI/UX Design Principles
* Responsive Web Design
* DOM Manipulation
* Form Integration (API-based)
* Basic Deployment (GitHub Pages)

---

## 📁 Project Structure

```
project-folder/
│── index.html     # Structure and content

```
